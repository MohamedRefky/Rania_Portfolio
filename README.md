## 🚀 Running the Project

To run this portfolio app locally:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/MohamedRefky/Rania_Portfolio.git
   ```
2. **Navigate to the directory:**
   ```bash
   cd Rania_Portfolio
   ```
3. **Install dependencies:**
   ```bash
   flutter pub get
   ```
4. **Run the application:**
   ```bash
   flutter run
   ```
